### Написать плагин для сайта на 1С-Битрикс.
```
1. Плагин должен фиксировать все данные, которые отправляются любой формой с любой страницы сайта. На каждое заполнение формы должен создаваться отдельный файл с собранными данными в виде JSON структуры.
2. В администраторской панели сайта необходимо реализовать выбор папки, в которую будут сохраняться файлы с данными.
3. Готовый код необходимо предоставить в виде архива для установки на сайт 1С-Битрикс.
```
